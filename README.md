🍽️ Rescue Plate - Food Rescue Platform
A full-stack web application that connects restaurants with surplus food to local NGOs and shelters. Built to fight hunger and reduce food waste.

🚀 Live Demo
Website: https://rescue-plate-vipin.netlify.app
API: https://rescue-plate-api-v2.onrender.com
🛠 Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript (Fetch API)
Backend: Node.js, Express.js
Database: MongoDB Atlas (Mongoose ODM)
Authentication: JWT (JSON Web Tokens) & Bcrypt.js
Hosting: Render (Backend), Netlify (Frontend)
✨ Features
User Authentication: Secure Signup/Login with password hashing.
Create Donations: Restaurants can post surplus food details.
View Donations: NGOs can view real-time available food pickups.
Protected Routes: Frontend access is restricted for logged-out users.
Dynamic UI: Navbar dynamically updates based on login state.
